# Financial Control App

### Pre requisites:

- [Docker](https://docs.docker.com/install/)
- [Docker Compose](https://docs.docker.com/compose/install/)

### Usage

- Build:

```bash
$ docker-compose up -d --build
```

- Setup:

```bash
$ docker-compose up -d
```

- Stop:

```bash
$ docker-compose down
```
